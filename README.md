# Vulnerable Login App

This is an educational project demonstrating a SQL injection vulnerability in a simple login system.

## Setup

1. Install dependencies: `npm install`
2. Run the server: `npm start`
3. Open http://localhost:3000 in your browser

## Usage

**Admin Account:**  
- Username: admin  
- Password: password  

**Normal User Account:**  
- Username: user  
- Password: pass  

Or exploit the SQL injection vulnerability.

## Note

This app contains intentional vulnerabilities for educational purposes. Do not use in production.
